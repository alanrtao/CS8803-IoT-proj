# CS8803-IoT-proj
Microcontroller for light condition syncing

## Setup
- Install Arduino IDE
  - In preference, add "https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json" to additional board manager URL
  - In tools/board manager, install ESP32 from Expressif.
  - Install [PID library](https://www.arduino.cc/reference/en/libraries/pid/)
